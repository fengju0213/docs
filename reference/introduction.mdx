# CAMEL-AI API Reference

Welcome to the comprehensive API reference for CAMEL-AI! 🐪

This section contains detailed documentation for all modules, classes, and functions in the CAMEL-AI library.

## Core Components

### 🤖 Agents
Multi-agent systems and individual agent implementations for various tasks.

### 🧠 Models  
Model integrations for different LLM providers and configurations.

### 🛠️ Toolkits
Extensive collection of tools to extend agent capabilities.

### 💾 Memory & Storage
Memory systems and storage solutions for persistent agent state.

### 📨 Messages
Message handling and conversation management systems.

### 🔍 Retrievers & Embeddings
Search and retrieval capabilities with various embedding models.
